`zjss` was retired from the active repo after repeated Zellij-session/layout issues and sizing side effects.

This directory keeps the old implementation and demo artifacts around in case we want to revisit it later.

Archive contents:
- `zjss-function.zsh`: snapshot of the old shell function removed from `.zshrc.template.sh`
- `zjss-wrapper.sh`: SSH helper used by the old layout
- `tools/`: old demo tape and fake layout
- `assets/`: old demo GIF
